Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => app/src/main/AndroidManifest.xml
* libs/android-support-v4-r13.jar => app/libs/android-support-v4-r13.jar
* libs/armeabi-v7a/libIndoorsLocator.so => app/src/main/jniLibs/armeabi-v7a/libIndoorsLocator.so
* libs/armeabi/libIndoorsLocator.so => app/src/main/jniLibs/armeabi/libIndoorsLocator.so
* libs/indoors-library-surface.jar => app/libs/indoors-library-surface.jar
* libs/x86/libIndoorsLocator.so => app/src/main/jniLibs/x86/libIndoorsLocator.so
* lint.xml => app/lint.xml
* res/ => app/src/main/res/
* src/ => app/src/main/java/
